yum源服务器的建立

时间: 2016-03-13 14:16:53     来源: 财务狗

直接连公共的yum服务器很慢。

我们可以建立一个私人的yum服务器

然后做30天的缓存。因为yum 和apt-get都是天天更新的。

只要30天不间断。那么速度会和内网速度差不多。


server {

listen 80;
server_name yum.caiwuhao.com;
location /{
proxy_pass   http://mirrors.aliyun.com;
#proxy_set_header host otherdomain.com';
#proxy_cache one;
#proxy_cache_valid  200 304  12h;
#proxy_cache_valid any 1m;
expires 30d;
}
}
