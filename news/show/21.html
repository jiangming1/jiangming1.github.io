自动化运维系列-自动化安装centos7

时间: 2016-03-15 23:34:08     来源: 财务狗

# automation

自动化运维.

小公司多的是运维的事情.电脑坏了呀.中病毒了等.那么我们需要把一些重复的工作自动化.免得让他们来打扰你

除此之外,需要学会拒绝.拒绝额外的负担.什么家里电脑坏了让你修一下呀.帮忙呀.不然忙死累死的就是你.

这个是我编辑的总结.学会了这些.系统装机就很简单了.再深入一点.也只不过是架设一台pxe服务器内网开启自动装机模式而已.这样pxe启动后就自动把系统给装好了. 

编辑isolinux.cfg 增加ks.cfg的路径

append  inst.ks=http://www.caiwuhao.com/ks7.cfg 

要把rd.live.check 删除掉

1.把ks.cfg文件放入光盘的根目录

2.把ks.cfg放到网络上

上传文件到服务器上

#version=DEVEL 
install  #表示是安装，而不是升级 
text #文本方式安装 
url --url=http://mirrors.aliyun.com/centos/7.2.1511/os/x86_64/
lang en_US.UTF-8  #语言 
keyboard us #键盘样式 
network --onboot yes --device eth0 --bootproto dhcp --noipv6
rootpw 123456 
firewall --disabled
authconfig --enableshadow --passalgo=sha512
selinux --disabled
timezone --utc Asia/Tokyo
bootloader --location=mbr --driveorder=sda --append="nomodeset crashkernel=auto rhgb quiet"
 
zerombr
clearpart --linux --drives=sda
autopart
 
%packages --nobase
 
@core                                                                           
%end
%post
useradd dev
useradd localadmin
echo '123456' | passwd --stdin root
echo '123456' | passwd --stdin localadmin
echo '123456' | passwd --stdin dev
%end
